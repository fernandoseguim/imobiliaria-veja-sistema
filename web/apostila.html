Apostila do Administrador de Web Site

Esta apostila contém todas as informações para você gerenciar o seu Web Site. Vamos mostrar passo a passo como trabalhar com as informações contidas no seu site de maneira fácil e rápida. Disponibilizando para os seus clientes uma interação bem produtiva.

Primeiro entre na página www.imobiliariaveja.com.br/senha.asp:

Você vai precisar de uma senha e um ID , no exemplo da apostila vou criar uma senha e um ID fictícios:

ID: entrar

Senha: entrar

Coloque a senha e o ID no formulário , observe que na caixa de texto da senha as letras vão ser digitadas como asterísticos ****** , isso é normal, se trata de um recurso do sistema para maior segurança. Depois clique em entrar.

Agora você já está no administrador do site como mostrado abaixo:

Agora nós vamos entender como funciona o administrador:

Veja quais as funções dos números indicativos:

1) O número 1são os links para os vários bancos de dados do site, como cidade , proposta, permuta, compradores, imóveis, mas vamos focar nos imóveis , se você souber gerenciar os imóveis saberá gerenciar o site inteiro.

2) O número 2 indica um site de busca de imóveis por cidade, bairro, negociação, quartos e valor, com esse buscador você poderá encontrar imóveis facilmente , com as características pedidas por seu cliente.

3) O número 3 indica um outro tipo de buscador usando proprietário, endereço , telefone, data, primeira página, standby e código do imóvel. Esse buscador é eficiente para buscar imóveis com os dados do proprietário e para administrar a primeira página e o standby.Nós vamos mostrar mais adiante tudo isso detalhadamente.

Deverá aparecer uma tela parecida com o mostrado abaixo:

Agora mostraremos o que significa os indicadores na tela:

1) O número 1 é o botão que exclui vários registros no banco de dados.

Os registros são as informações que você vê na lista apresentada na tela que contêm dados do proprietário , endereço do imóvel e data, são as informações referentes aos imóveis.

2) O número 2 é um botão que vai abrir um formulário para a inclusão de um novo imóvel.

3) O número 3 é o botão que exclui apenas um imóvel do banco de dados.

4) O número 4 é o botão que abre um formulário de visualização dos dados do imóvel, lá você poderá ver todos os dados que o imóvel contém e também poderá atualiza-los se quiser.

5) O número 5 é o botão indicando se o imóvel tem foto ou não, quando tem essa maquininha fotográfica no registro significa que o imóvel tem foto, e para visualizar essa foto é só passar o mouse por cima da maquininha que o imóvel será mostrado. Esse botão também serve para visualizar os dados do imóvel e sua possível atualização.

Agora vamos aprender a incluir um imóvel com foto!!

Vá no administrador e clique no botão incluir para abrir o formulário de inclusão de imóveis.

Veja que abriu o formulário de inclusão, agora clique no botão cadastramento com foto.

O formulário de cadastramento com foto parece igual ao formulário comum, mas note as caixas de texto de Foto Grande e Foto Pequena.

Você clicando no botão procurar da caixa de texto de foto grande e foto pequena abrirá para você uma janela onde você poderá procurar uma foto nas pastas do seu computador.

É só procurar o arquivo de foto e clicar 2 vezes em cima dele e automatimente o endereço do arquivo vai para a caixa da foto.

Pronto agora é só preencher as outras caixas de texto e temos a inclusão no banco de dados de um imóvel com foto.

Para excluir um imóvel com foto você faz a mesma coisa que fez para excluir um imóvel sem foto.

Obs: A caixa de texto foto pequena se refere a foto que vai aparecer na lista de imóveis procurados ela deve ser igual a foto grande.Dê preferencia tenha duas fotos iguais do imóvel mas de tamanhos diferentes. Para que a foto pequena seja visualizada mais rapidamente.

Recomendamos os seguintes tamanhos para as fotos dos imóveis.

Foto Grande: 400 x 230 pixels

Foto Pequena: 158 x 90 pixels
